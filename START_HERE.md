## 🚀 立即开始（3 步）

### 步骤 1: 安装依赖
```bash
cd E:/trend-rush-skill
npm install
```

### 步骤 2: 配置密钥
```bash
cp config.example.json config.json
# 编辑 config.json 添加你的 API 密钥
```

### 步骤 3: 运行第一个项目
```bash

claude skill trend-rush --keyword="AI写作工具" --mode=auto

# 方式 B: 手动运行
npm run full
```
